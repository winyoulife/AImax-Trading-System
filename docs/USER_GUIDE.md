## 🚀 快速開始

### 第一次使用

1. **訪問您的AImax系統**
   - 打開瀏覽器，訪問您的GitHub Pages網址
   - 格式: `https://您的用戶名.github.io/倉庫名/`

2. **安全登入**
   - 輸入您在部署時設置的專屬帳號和密碼
   - 系統採用SHA-256加密保護您的帳號安全
   - 會話有效期為24小時

3. **探索儀表板**
   - 查看即時BTC/TWD價格
   - 監控交易系統運行狀態
   - 檢查GitHub Actions執行情況

## ⚙️ 系統配置

### GitHub Secrets 配置

訪問: `https://github.com/您的用戶名/倉庫名/settings/secrets/actions`

**必需配置**:
```
ADMIN_USERNAME: 您的登入帳號
ADMIN_PASSWORD_HASH: 自動生成的密碼哈希
```

**交易配置** (如需真實交易):
```
MAX_API_KEY: MAX交易所API密鑰
MAX_API_SECRET: MAX交易所API密鑰密碼
```

**Telegram通知** (可選):
```
TELEGRAM_BOT_TOKEN: Telegram Bot Token
TELEGRAM_CHAT_ID: Telegram Chat ID
```

## 📈 交易策略說明

### 83.3%勝率MACD策略

**核心邏輯**:
1. **趨勢識別**: 使用MACD指標識別市場趨勢
2. **信號確認**: 多重條件確認交易信號
3. **風險控制**: 智能止損和資金管理
4. **頻率調整**: 根據市場波動性調整執行頻率

**執行條件**:
- **買入信號**: MACD線上穿信號線且兩線均為負值
- **賣出信號**: MACD線下穿信號線且兩線均為正值
- **風險控制**: 設置止損和止盈點
- **資金管理**: 每次交易不超過總資金的一定比例

### 智能頻率控制

**高波動期** (價格變化 > 2%):
- 台灣交易時段: 每1分鐘執行
- 其他時段: 每3分鐘執行

**中等波動期** (價格變化 0.5-2%):
- 台灣交易時段: 每3分鐘執行
- 其他時段: 每5分鐘執行

**低波動期** (價格變化 < 0.5%):
- 台灣交易時段: 每5分鐘執行
- 其他時段: 每15分鐘執行
